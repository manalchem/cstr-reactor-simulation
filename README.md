# CSTR Reactor Simulation — Ethylene Oxide Production

A chemical engineering simulation built in Python and Google Colab.

## Overview
Models a Continuous Stirred Tank Reactor (CSTR) for ethylene oxide production using:
- Arrhenius equation for temperature-dependent rate constants
- CSTR design equation for first-order reactions
- Conversion vs. residence time analysis at 400K, 450K, and 500K

## Key Results
| Temperature | Rate Constant k | Conversion X | CA (mol/L) |
|-------------|----------------|--------------|------------|
| 400K | 0.01606 s⁻¹ | 7.4% | 0.9257 |
| 450K | 0.19675 s⁻¹ | 49.6% | 0.5041 |
| 500K | 1.46057 s⁻¹ | 87.9% | 0.1204 |
